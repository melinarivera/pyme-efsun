# La Mágica Bruja Efsun ✨

Sitio web completo para un negocio de servicios esotéricos. Incluye tienda online, sistema de reservas, blog y horóscopo generado por inteligencia artificial.

## Demo

> Single-page application — abre `index.html` directamente en el navegador.

## Funcionalidades

- **Horóscopo con IA** — Lecturas personalizadas generadas por Claude (Anthropic) según signo zodiacal y área de vida
- **Reservas online** — Flujo de 4 pasos para agendar sesiones presenciales u online
- **Tienda** — 20 productos esotéricos organizados en 9 categorías con carrito de compras
- **Servicios** — 6 servicios (Tarot, Videncia, Limpia energética, Rituales, Magia, Pareja)
- **Blog** — Artículos sobre luna, astrología, tarot y rituales con sistema de comentarios

## Tecnología

| Capa | Detalle |
|------|---------|
| Frontend | HTML + CSS + JavaScript vanilla |
| IA | Claude Sonnet (Anthropic API) |
| Build | Sin dependencias — archivo único `index.html` |
| Deploy | Cualquier hosting estático |

## Uso local

```bash
# Clona el repositorio
git clone https://github.com/tu-usuario/pyme-efsun.git

# Abre en el navegador
open index.html
```

Para el horóscopo con IA necesitas una API key de [Anthropic](https://console.anthropic.com/).
